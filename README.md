# Gumi Smart Learning Support Service

A fully web-based smart learning platform designed to help students study more effectively using AI.

## Features

- **Six Major Subjects**: Korean, Math, English, Social Studies, Science, Korean History
- **Three Difficulty Levels**: High, Medium, Low
- **AI Learning Assistant**: 24/7 AI tutor for personalized learning support
- **7-Day AI Prompting Course**: Teaches effective AI communication skills
- **Bilingual Support**: Full Korean and English language support
- **Personalized Learning Paths**: Different paths for general and advanced students
- **Responsive Design**: Works on desktop, tablet, and mobile devices
